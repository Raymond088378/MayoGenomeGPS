#### INPUT file format

1. alignment/mayo/external module

FASTQ files (if zipped, file name should end with .gz)

2. realignment/realign-mayo module

aligned BAM files 

3. variant module

realigd BAM files

4. ontarget module

RAW VCF variant files (only exome)

5. annotation module  (positions should be 0 based)

VCF format file (SNV or INDEL or BOTH)

TXT format file  (close to vcf format, just use column 1,2,4,5 from vcf to create the text format)

SNV (four columns chromosome, position, Ref Allel, Alt Allele )
*** no header
*** chr should be chr1 etc.

coordinate system we are following is the VCF format 4.1
example

chr1	100	A	T
chr1	200	C	T
.....

INDEL (five columns : chromsome, pos, RefAllele, Alt Allele)
*** no header
*** chr should be chr1 etc.
example

chr1	100	A	AT
chr1	200	ATC	A
......
